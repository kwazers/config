## 🛠 Инструкция по установке

### Способ 1: Ссылка для подписки (Рекомендуемый)
1. Скопируйте одну из ссылок `https://raw.githubusercontent.com/...` выше.
2. Откройте ваш VPN-клиент (**Hiddify, v2rayNG, Sing-box, Nekobox** или **Happ**).
3. Перейдите в раздел **Группы / Подписки / Профили** (Groups/Subscriptions).
4. Нажмите **Добавить (+)**, введите любое имя и вставьте скопированную ссылку в поле URL.
5. Нажмите кнопку **Обновить подписку**. Приложение автоматически загрузит рабочие ключи.

### Способ 2: Через QR-код
1. Откройте приложение VPN на смартфоне.
2. Нажмите кнопку **Сканировать QR-код**.
3. Наведите камеру на QR-код нужной страны на этой странице. Профиль импортируется мгновенно.
