Пинг серверов через proxy (от лица пользователя страницы)

В поле ниже укажите URL proxy, который умеет проксировать произвольные URL и поддерживает CORS. Примеры: https://your-proxy.example.com/ или https://cors-anywhere.herokuapp.com/ (требует активации). Если proxy содержит токен подстановки {url}, он будет заменён на адрес цели (например: https://proxy/?u={url}).
Список целей (можно добавить свои через кнопку «Добавить»)
Результат будет сохранён в файл (скачивается клиенту). Имя папки определяется локально по IP (в имени файла).
Примечание: успешный результат зависит от возможностей указанного proxy и CORS. Если proxy не поддерживает подстановку полных URL, используйте прокси с шаблоном {url} или тот, который принимает полный URL в конце пути.
